===========================================================================
        SERIOUSPRENUER — PAID API RECOMMENDATIONS
        (When Ready to Scale Beyond Free Tier)
===========================================================================

Created: 2026-03-27
Status: MVP using all FREE APIs. Upgrade when metrics justify spend.

---------------------------------------------------------------------------
1. LEAD DISCOVERY — Google Places API
---------------------------------------------------------------------------
FREE:  $200/month credit (covers ~5,000 Nearby Search requests)
PAID:  $17 per 1,000 Basic requests / $32 per 1,000 Advanced requests
WHEN:  When you exceed 5,000 searches/month or need detailed business info
NOTE:  Consider SerpAPI ($50/mo) as alternative for richer data

---------------------------------------------------------------------------
2. AI OUTREACH — Google Gemini API
---------------------------------------------------------------------------
FREE:  Gemini 1.5 Flash — 15 RPM, 1M tokens/min, 1.5B tokens/day
PAID:  Gemini 1.5 Pro — $3.50 per 1M input tokens, $10.50 per 1M output
WHEN:  When you need higher quality outputs or exceed rate limits
ALT:   OpenAI GPT-4o-mini at $0.15/$0.60 per 1M tokens (input/output)
ALT:   Anthropic Claude 3.5 Haiku at $0.25/$1.25 per 1M tokens

---------------------------------------------------------------------------
3. WEBSITE SEARCH CHECK — Google Custom Search Engine (CSE)
---------------------------------------------------------------------------
FREE:  100 searches/day (3,000/month)
PAID:  $5 per 1,000 queries beyond free tier
WHEN:  When daily search volume exceeds 100
ALT:   Bing Web Search API — 1,000 free/month, then $3/1,000 calls

---------------------------------------------------------------------------
4. DATABASE — Supabase
---------------------------------------------------------------------------
FREE:  500MB database, 1GB file storage, 50K monthly active users
PAID:  Pro @ $25/month — 8GB database, 100GB storage, daily backups
WHEN:  When database exceeds 500MB or you need backups/point-in-time recovery
NOTE:  Free tier is generous for MVP; upgrade around 200-500 active users

---------------------------------------------------------------------------
5. WORKFLOW AUTOMATION — n8n.io
---------------------------------------------------------------------------
FREE:  n8n Cloud Starter — 5 workflows, 2,500 executions/month
PAID:  n8n Pro @ €24/month — Unlimited workflows, 10K executions
SELF:  Self-host on Railway/Render for ~$7/month (unlimited everything)
WHEN:  When you exceed 2,500 monthly workflow executions
NOTE:  Self-hosting is most cost-effective for scaling

---------------------------------------------------------------------------
6. WEBSITE DEPLOYMENT — Vercel
---------------------------------------------------------------------------
FREE:  100GB bandwidth, 6,000 build minutes/month, serverless functions
PAID:  Pro @ $20/month — 1TB bandwidth, 60K build minutes
WHEN:  When deploying 50+ client sites or bandwidth exceeds 100GB
ALT:   Netlify (same pricing), Cloudflare Pages (unlimited free bandwidth)

---------------------------------------------------------------------------
7. WhatsApp BUSINESS API — Meta/Twilio
---------------------------------------------------------------------------
FREE:  None (Meta requires Business Manager + $0.005-$0.08 per message)
PAID:  Twilio WhatsApp — $0.005/message + $0.005 carrier fee
WHEN:  Phase 2 — when you want automated WhatsApp sending
ALT:   Start with manual WhatsApp copy-paste in MVP (FREE)

---------------------------------------------------------------------------
8. EMAIL SENDING — SendGrid / Resend
---------------------------------------------------------------------------
FREE:  SendGrid: 100 emails/day | Resend: 100 emails/day
PAID:  SendGrid Pro @ $19.95/month (50K emails) | Resend Pro @ $20/month
WHEN:  When automated email outreach exceeds 100/day
NOTE:  MVP uses copy-to-clipboard; upgrade to auto-send in Phase 2

---------------------------------------------------------------------------
9. SMS — Twilio
---------------------------------------------------------------------------
FREE:  Trial account with $15 credit (~$0.08 per SMS in India)
PAID:  Pay-per-use: ~$0.04/message (India) + $1/month for phone number
WHEN:  Phase 2 — when SMS outreach becomes a feature
ALT:   MSG91 (India-specific, cheaper for Indian numbers)

---------------------------------------------------------------------------
10. ANALYTICS — PostHog / Mixpanel
---------------------------------------------------------------------------
FREE:  PostHog: 1M events/month | Mixpanel: 20M events/month
PAID:  PostHog Growth @ $0.00031/event beyond 1M
WHEN:  When you need detailed user behavior analytics
NOTE:  Google Analytics 4 is always free and sufficient for MVP

---------------------------------------------------------------------------
11. IMAGE/ASSET GENERATION — Unsplash / Pexels API
---------------------------------------------------------------------------
FREE:  Unsplash: 50 requests/hour | Pexels: 200 requests/hour
PAID:  Shutterstock API @ $29/month for 10 images/month
WHEN:  When you need premium stock images for generated websites
NOTE:  Unsplash/Pexels free tiers are sufficient for MVP

---------------------------------------------------------------------------
12. DOMAIN MANAGEMENT — Cloudflare
---------------------------------------------------------------------------
FREE:  DNS management, basic SSL, DDoS protection
PAID:  Domain registration @ $8-15/year per domain
WHEN:  When offering custom domains to clients
NOTE:  MVP uses subdomains (free); custom domains in Phase 3

===========================================================================

ESTIMATED Phase 2 COST: ~$50-75/month
ESTIMATED Phase 3 COST: ~$150-200/month
